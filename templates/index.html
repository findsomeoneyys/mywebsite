How It Works

Set Settings

The Scrapy settings allows you to customize the behaviour of all Scrapy components, including the core, extensions, pipelines and spiders themselves.

Start Spider

In the project directory of the command line, enter scrapy crawl news to start the project.

Automatically analyze news pages and use the Natural Language Toolkit to process word segmentation, get information, and store it in mongo database
